Создание локального Сайта.
=========================================================================================
1. зарегистрироваться на GitHub.com
2. Создать репозиторий - Нажимаем + (плюс) в правом верхнем углу сайта - и New repository
3. вводим : Repository name - Пишем : NameRep.Github.io
4. Добавляем : Description + Initialize this repository with a README
5. Открываем браузер и вводим адрес : NameRep.Github.io
6. Открыть файл - Readmy.md - и заменить NameRep.Github.io на свое имя
7. и Написать : Мои домашки
8. Добавить строку : [Сайт №1](NameRep.Github.io/Site1 "Мой первый сайт!")
9. Сохранить изминения.
10. На рабочем столе создать папку GitHub а в ней папку Site1
11. В папку Site1 - скопировать свои файлы
12. Нажать на GitHub.com - Upload
13. Скопировать все файлы на сайт

1. Склонировать проект себе на ubuntu в папку Site1 
	git clone git://github.com/Rymar4/html.git html

2. Прописать данные о себе чтобы авторы проекта знали кто вносил изминения
	git config --global user.name "Rymar Viktor"
	git config --global user.email "rymar@ukr.net"

3. Создаем свою ветку и перейдем в нее
	git checkout -b new1
 
4. Смотрим какая ветка выбрана
	git branch

5. Делаем изменения - Site-Lab

5. смотрим изминения - 
	git status

6. Добавляем изминения в репозиторий
	git add *.php

7. Фиксируем изминения
	git commit -m 'Изминение №1'
	
9. Теперь перейдем в ветку мастер :
	git checkout master

10. Соединяем нашу работу с основным проектом
	git merge new1

11. Связываем свой репозиторий с GitHub
	git remote add rip1 https://github.com/............

8. После всей работы - мы сливаем в мастер
	git push -u rip1 master

12. Удаляем ветку new1
	git brench -D new1

Создание нового репозитория в существующем каталоге - 

1. Перейти в проектный каталог и в командной строке ввести 
	git init
	
2. Прописать данные о себе чтобы авторы проекта знали кто вносил изминения
	git config --global user.name "Rymar Viktor"
	git config --global user.email "rymar@ukr.net"

3. Создать файл new.txt - и внести в него текст "Моя первая запись!!" 

4. Проиндексировать эти файлы и осуществить первую фиксацию изменений.
	если ввести (git add .) то все закрепится
	git add new.txt

5. Фиксация изменений
	git commit -m 'Коментарий - Первое изминение!'

6. Посмотреть что сделано
	git status
	
7. Создаем удаленный репозиторий на GitHub - rip2

8. Добавить remote - сопоставление или связывание локального имени origin (может быть свое)
	с удаленным репозиторием на GitHub
	git remote add rip2 https://github.com/rymar4/rip2.git	
	
9. Смотрим какие удаленные репозитории есть -
	git remote 	

10. Сделать push - залить файлы на GitHub в rip2 c ветки master 
	git push -u rip2 master
