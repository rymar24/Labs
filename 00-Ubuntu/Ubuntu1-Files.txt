Переключиться на одну из виртуальных консолей можно нажав сочетание клавиш:
Ctrl+Alt+F1 - первая виртуальная консоль;
Ctrl+Alt+F2 - вторая виртуальная консоль;
Ctrl+Alt+F3 - третья виртуальная консоль;
Ctrl+Alt+F4 - четвертая виртуальная консоль;
Ctrl+Alt+F5 - пятая виртуальная консоль;
Ctrl+Alt+F6 - шестая виртуальная консоль;
Ctrl+Alt+F7 - седьмая виртуальная консоль, возврат в графический режим.

После запуска терминала мы видим строку с приглашением к вводу команд, например:
vladimir@Zotac-Zbox-Nano:~$

vladimir - имя учетной записи пользователя
@ - разделитель между учетной записью и именем компьютера
Zotac-Zbox-Nano - имя компьютера
: - разделитель
~ - в какой папке выполняется команда, ~ это домашняя папка пользователя, если выполните команду ls то получите список файлов из этой паки
$ - приглашение к выполнению команды с правами простого пользователя (# будет означать приглашение на выполнение команд с правами администратора)

==========================================  Работа с файлами ======================================
Зайти в домашний каталог ( cd ~ )
Создать каталог cat ( mkdir cat )
Перейти в cat (cd cat)
Создать в cat  три директории одновременно dir1 dir2 dir3 (mkdir dir1 dir2 dir3)  - проверить ( ls -l )
Установить программу tree (sudo apt install tree)
Создать дерево директорий ~/cat/dir1/cat1/files1  (mkdir -p ~/cat/dir1/cat1/files1)  - проверить ( tree )
------------------------------ Показать -----------------------------------------------------------
Перейти в каталог dir1 ( cd ~/cat/dir1 )
Создать file1.txt в dir1 ( touch file1.txt )  - проверить ( ls -l )
Направить стандартный ввод в file1.txt в каталоге dir1 (cat > file1.txt)
ввести текст - цифры в столбик от 1 до 20 , после окончании ввода нажать Ctrl+d) 
Вывести содержимое file1.txt (more file1.txt )
Вывести содержимое файла file1.txt (less file1.txt) выйти с редактора - нажать Q
Вывести первые 10 строк file1.txt (head file )
Вывести последние 10 строк file1.txt (tail file)

Перейти в домашний каталог ( cd ~ )
Перейти в директорию двумя уровнями выше (cd ../..)
Перейти снова в домашнюю директорию (cd ~ )
Перейти в директорию, в которой находились до перехода в текущую директорию (cd - )
Перейти в домашнюю директорию пользователя user  (cd ~user)
Показать текущюю директорию (pwd)

Перейти в каталог dir1 ( cd ~/cat/dir1 )
Создать файл file2.txt  с текстом Hello World (echo "Hello World" > file2.txt)
Скопировать file2.txt в file3.txt (cp file2.txt file3.txt)
Удалить file3.txt (rm file3.txt)  -   - проверить ( ls -l )

Перейти в каталог cat ( cd ~/cat )
Удалить директорию с именем 'dir3' и рекурсивно всё её содержимое (rm -rf dir3 )

Перейти в домашний каталог ( cd ~ )
Создать дерево директорий ~cat/dir1/cat1/papka1  (mkdir -p ~/cat/dir1/cat1/papka1) - проверить ( tree )
Создать дерево директорий ~cat/dir2/cat2/papka2  - командой mkdir -p cat/dir2/cat2/papka2 - проверить ( tree )
Создать дерево директорий ~cat/dir3/cat3/papka3  - командой mkdir -p dir3/cat3/papka3 - проверить ( tree )

Cкопировать dir1 в dir4; создаст каталог dir4, если он не существует (cp -r dir1 dir4)  - проверить ( tree )

Перейти в каталог dir4 (cd ~/cat/dir4)
Скопировать все файлы директории /var/log/apt в текущую директорию (cp -a /var/log/apt dir4)
Скопировать директорию dir1 в директорию dir2 (cp -a dir1 dir2)
Переименовать или переместить dir4 в dir2 если dir2 существующий каталог (mv dir4 ../dir2) 
Создать символическую ссылку на файл file1.txt в каталоге dir1 (ln -s file1.txt lnk1)
В домашнем каталоге создать символическую ссылку на каталог papka1 (ln -s ~/cat/dir1/cat1/papka1 pk1)
