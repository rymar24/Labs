
=============================   Пользователи и группы ============================================
Cоздать пользователя user1 user2 user3 user4 (sudo adduser user1)
Удалить пользователя user4 и его домашний каталог (sudo userdel -r user4)
Сменить пароль  user1 (sudo passwd user1)

Создать новую группу с именем group1 и group2 (sudo groupadd group1)
Посмотреть списог груп и найти созданную (sudo cat /etc/group | more)
Удалить группу group2 (sudo groupdel group2)

================================   Выставление/изменение полномочий на файлы =====================================

Зайти пользователем user1 - нажать Alt+F2 и ввести логин и пароль user1
Создать папку shara (mkdir shara)
Показать полномочия на файлы и директории в текущей директории (ls -lh)

Зайти пользователем user2 - нажать Alt+F3 и ввести логин и пароль user2
проверить доступ к папки shara (ls -l /home/user1)
перейти в каталог shara (cd /home/user1/shara)
Попробовать создать файл - file1.txt (touch file1.txt)

Перейти к пользователю user - нажать Alt+F1
Для выполнения команд с правами root (Администратора)
 добавить пользователя user1 в группу sudo - (sudo usermod -a -G sudo user1)
 добавить пользователя user2 в группу sudo - (sudo usermod -a -G sudo user2)

Занести пользователей user1 user2 в одну группу group1
 добавить пользователя user1 в группу group1 - (sudo usermod -a -G group1 user1)
 добавить пользователя user2 в группу group1 - (sudo usermod -a -G group1 user2)

Выйти и снова зайти пользователем user2 - нажать Alt+F3 набрать exit и снова ввести логин и пароль user2
перейти в каталог shara (cd /home/user1/shara)
Попробовать создать файл - file1.txt (touch file1.txt)
Проверит создан ли файл (ls -l)

Зайти пользователем user3 - нажать Alt+F3 и ввести логин и пароль user3
перейти в каталог shara (cd /home/user1/shara)
Попробовать создать файл - file1.txt (touch file1.txt)

Перйти к пользователю user1
Добавить полномочия на директорию shara ugo(User Group Other)+rwx(Read Write eXecute) - всем полные права. (sudo chmod ugo+rwx shara)

Перйти к пользователю user2 нажать Alt+F3
проверить достук к каталогу shara - Попробовать создать файл - file1.txt (touch file1.txt)

Перйти к пользователю user1
Отобрать у группы и всех остальных все полномочия на директорию shara (sudo chmod go-rwx dir1)

Популярные значения
400 (-r——–) 	Владелец имеет право чтения; никто другой не имеет права выполнять никакие действия
644 (-rw-r–r–) 	Все пользователи имеют право чтения; владелец может редактировать
660 (-rw-rw—-) 	Владелец и группа могут читать и редактировать; остальные не имеют права выполнять никаких действий
664 (-rw-rw-r–) 	Все пользователи имеют право чтения; владелец и группа могут редактировать
666 (-rw-rw-rw-) 	Все пользователи могут читать и редактировать
700 (-rwx——) 	Владелец может читать, записывать и запускать на выполнение; никто другой не имеет права выполнять никакие действия
744 (-rwxr–r–) 	Каждый пользователь может читать, владелец имеет право редактировать и запускать на выполнение
755 (-rwxr-xr-x) 	Каждый пользователь имеет право читать и запускать на выполнение; владелец может редактировать
777 (-rwxrwxrwx) 	Каждый пользователь может читать, редактировать и запускать на выполнение
1555 (-r-xr-xr-t) 	Каждый пользователь имеет право читать и запускать на выполнение; удалить файл может только владелец этого файла
2555 (-r-xr-sr-x) 	Каждый пользователь имеет право читать и запускать на выполнение с правами группы(user group) владельца файла
4555 (-r-sr-xr-x) 	Каждый пользователь имеет право читать и запускать на выполнение с правами владельца файла
